#Odin Recipes

##Background
This is my implementation of the first project in the web development curriculum compiled by [The Odin Project](https://www.theodinproject.com/). The lessons revisit this project multiple times, starting with HTML and later styling the pages using CSS.

##Goals
The goal of this project is to learn the basics of HTML and CSS, and to gain experience organizing the source pages and assets that make up a static web site. A secondary goal is to gain experience managing a web development project with Git.
